# Home Module

This directory contains the main dashboard page and its specific components.

## Components

### Main Widgets
-   **ConnectionSuggestions**: Shows suggested user connections.
-   **RemindersBox**: Lists upcoming academic reminders.
-   **PrintHistoryBox**: Shows recent print jobs.

### Helper Components
-   **PostCard**: Individual post display card.
-   **NewPost**: Modal for creating/editing posts.
-   **ConnectionCard**: Individual connection display card.
-   **ConnectionCardSkeleton**: Loading state for connection cards.
-   **ImageModal**: Modal for viewing post images.

## Structure

-   `components/`: Local components used only by the Home page.
-   `Home.tsx`: The main page component assembling the dashboard.
